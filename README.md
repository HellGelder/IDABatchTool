 # IDA Batch Tool
 
  Возможности
 
 * Пакетный анализ файлов с параллельным запуском IDA
 * CLI и графический интерфейс на PySide6
 * Автоопределение разрядности и архитектуры (PE/ELF)
 * Экспорт дизассемблера и псевдокода в JSON
 * Интерактивные HTML‑отчёты с поиском и классификацией модулей
 * Встроенный классификатор более 1000 системных библиотек
 * Тёмная и светлая темы оформления
 
  Установка
 
 1. Клонируйте репозиторий
 2. Установите зависимости: `pip install -r requirements.txt`
 3. Настройте путь к IDA в `config.yaml`
 
  Быстрый старт (CLI)
 
 `python main.py --inputdir ./samples --analyse --filter ".exe,.dll"`
 
  Графический интерфейс
 
 `python main_gui.py`
 
  Конфигурация
 
 Файл `config.yaml` содержит пути утилитам IDA, количество потоков, тему
 
  Структура проекта
 
 * `ida_batch_tool/config` – загрузка конфигурации
 * `ida_batch_tool/discovery` – поиск файлов
 * `ida_batch_tool/ida` – управление запуском IDA
 * `ida_batch_tool/classifier` – классификатор модулей
 * `ida_batch_tool/reporting` – генератор отчётов
 * `ida_batch_tool/ui` – графический интерфейс
